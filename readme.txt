== Description ==

Layers for Publishers is a plugin that help you to insert the javascript from http://publishers.layers.com/ directly in your wordpress header


== Installation ==
1. Extract the wp-layers-for-publishers/ folder file to /wp-content/plugins/
1. Activate the plugin at your blog's Admin -> Plugins screen


== Frequently Asked Questions ==


== Usage ==
1. Click on Layers for Publishers on the main menu.
2. Insert Your layer script and save it
